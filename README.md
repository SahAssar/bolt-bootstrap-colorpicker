Bootstrap colorpicker
======================

A replacement for the colorpicker extension that no longer seems to work. Based on http://mjolnic.com/bootstrap-colorpicker/ and Raistlfiren/FAIconFieldType

Field Usage 

	icon:
  		type: colorpicker

Supported options:

	icon:
  		type: colorpicker
  		format: inline
  		hidden: true
        horizontal: true
        inline: true
        align: right
        hidden: true
